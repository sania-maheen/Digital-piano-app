# Digital-piano-app
A responsive digital piano web app built with HTML, CSS, and JS — created as part of my web development learning journey.
## Features
-> Interactive piano keys (white and black)
-> Plays sound for each note
-> Responsive and beginner-friendly design
-> Built using only frontend technologies

__________

## Technologies Used
-> HTML5 – structure of the web page  
-> CSS3 – styling and layout  
-> JavaScript (ES6) – interactivity and sound functionality  

__________

##  About This Project
This project was created as part of my learning practice from a Udemy course on web development.  
The original concept and base code were provided for educational purposes, and I modified it for practice and better understanding.
